# INSECTILES VISUAL DESIGN GUIDE

## Google Stitch Prompts

Use these prompts in **stitch.withgoogle.com** to generate your game UI:

### 1. MAIN MENU SCREEN
```
Sci-fi cyberpunk game menu with title "INSECTILES" in large neon green glowing text, dark black background with subtle grid pattern, futuristic buttons with green borders, "START", "CLASSES", "OPTIONS", "QUIT" menu buttons with hover glow effects, decorative insect/tech aesthetic, minimalist cyberpunk interface
```

### 2. CLASS SELECTION SCREEN
```
Game character selection screen showing 4 insect warrior options displayed as cards, each card shows: icon/emoji representation, character name, role (Striker/Tank/Support/Assassin), stat bars (Health Speed Damage Shield), sci-fi cyberpunk styling with neon green/red/gold accents, dark background with tech grid, selected state with glow
```

### 3. DIFFICULTY SELECTION
```
Game difficulty selection with 3 options: EASY NORMAL HARD displayed as futuristic panels, each shows enemy count, wave count, boss difficulty, sci-fi cyberpunk styling with color coding (green/yellow/red), dark tech background
```

### 4. GAMEPLAY HUD
```
Game interface showing: health bar (red gradient), XP bar (green gradient), shield bar (blue gradient) in top left, timer display center top, wave counter top right, score and kills counter, ability bar at bottom center, dark semi-transparent overlay, minimal HUD elements, cyberpunk sci-fi aesthetic
```

### 5. BOSS BATTLE HUD
```
Boss health bar at top of screen with boss name, large red health bar with gradient, phase indicator below, warning text area in center of screen for boss alerts, aggressive red cyberpunk styling
```

### 6. LEVEL UP SCREEN
```
Game level up screen showing 3 upgrade cards to choose from, each card displays: upgrade icon, upgrade name, description, stats change, gold/cyan styling, dark background, futuristic selection UI
```

### 7. GAME OVER SCREEN
```
Game over screen showing final score, waves survived, enemies killed, high score comparison, "PLAY AGAIN" and "MAIN MENU" buttons, red/dark cyberpunk aesthetic with dramatic styling
```

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Neon Green | #00ff88 | Primary accent, buttons, highlights |
| Dark Black | #020408 | Background |
| Cyber Red | #ff4444 | Health, danger, boss |
| Gold | #ffd700 | Level up, achievements |
| Cyan | #55aaff | Shields, info |
| Purple | #aa66ff | Synergy effects |

## Typography

- **Primary:** Orbitron (futuristic, bold)
- **Secondary:** Share Tech Mono (technical, readable)

---

## Export Settings

After generating in Google Stitch:
1. Click **Export** button
2. Choose **HTML + CSS** format
3. Copy code to `src/ui/` folder
4. Refine with OpenCode
